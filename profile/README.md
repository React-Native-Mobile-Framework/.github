## What is React Native?

React Native is an open-source framework for building native mobile applications using JavaScript and React . It was created by Facebook (now Meta) and first released in 2015 . React Native lets you build truly native apps with the power of React components, allowing you to write once and deploy to both iOS and Android .

Unlike hybrid frameworks that wrap web views, React Native compiles to native components, giving your app the feel and performance of a genuine native application . It has been used to build widely known apps such as Instagram, Facebook, and Airbnb .

Infrastructure teams building mobile applications benefit from React Native's ability to share business logic and most UI code between platforms, speeding up development and reducing maintenance costs . System administrators appreciate the hot-reloading feature, which drastically improves the developer experience by allowing developers to see changes immediately .

## Key Features

| Feature | Description |
|---------|-------------|
| **Cross-Platform** | Write once, deploy to iOS and Android (and Windows, macOS via community) |
| **Native Components** | Compiles to genuine native UI components, not web views |
| **Hot Reloading** | See changes instantly without recompiling |
| **React Ecosystem** | Full compatibility with React, JSX, React Hooks, and React state management |
| **Expo Integration** | Build apps without native iOS/Android SDKs with Expo Go |
| **Third-Party Libraries** | Extensive ecosystem of native modules (maps, analytics, payments) |
| **Developer Tools** | Flipper, React DevTools, and Chrome DevTools integration |
| **Native APIs Access** | Camera, GPS, notifications, Bluetooth, and more |
| **TypeScript Support** | Full TypeScript support for type-safe development |
